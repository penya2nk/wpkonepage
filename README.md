# wpkonepage
wpkonepage adalah tema WordPress satu halaman untuk website bisnis/profil perusahaan.
Dikembangkan dengan teknologi WordPress, PHP, MySQL, HTML5, CSS3, Javascript, jQuery, AJAX, Bootstrap 4, dan Font Awesome.
